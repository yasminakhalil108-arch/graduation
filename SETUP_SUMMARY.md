## 🛠️ طريقة الاستخدام

### لتفعيل صندوق الدردشة:

```html
<!-- في الـ HTML -->
<script src="ai-assistant-api.js"></script>
```

### في الكود:

```javascript
// 1. تهيئة الجلسة
AIAssistant.initSession();

// 2. معالجة السؤال
const response = AIAssistant.processQuery(userMessage);

// 3. حفظ الرسالة
await AIAssistant.saveMessage(userMessage, response);

// 4. تسجيل الحدث
AIAssistant.logEvent('message_sent', {});
```

## 🔄 دورة حياة الرسالة

```
المستخدم يكتب في صندوق الدردشة
         ↓
dونقر الزر أو يضغط Enter
         ↓
sendMessage() يتم استدعاؤها
         ↓
AIAssistant.processQuery() - بحث ذكي عن الإجابة
         ↓
إضافة مؤشر التحميل (800ms)
         ↓
عرض الإجابة على الشاشة
         ↓
AIAssistant.saveMessage() - حفظ في localStorage/Supabase
         ↓
AIAssistant.logEvent() - تسجيل الحدث
         ↓
تفعيل الحقل للسؤال التالي
```
